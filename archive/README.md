# Archive Folder Organization

This folder contains historical files from the MeeMaw Recipe website development process.

## Folder Structure

### `/conversion_scripts`
Python and PowerShell scripts used to process and convert recipe data during development:
- Recipe extraction scripts
- JSON conversion utilities
- Data comparison and validation tools

### `/backups`
Backup copies of recipe data and code from various stages of development:
- Original verbose JSON format (11,058 lines)
- Previous recipe data versions
- Code backups before major refactoring

### `/documentation`
Development documentation, templates, and notes:
- Recipe format templates
- Script usage documentation
- Project development summaries
- Original cookbook text file

### `/old_versions`
Obsolete or replaced JavaScript files:
- Previous script.js versions
- Experimental implementations
- Deprecated converters

## Active Project Files

The following files are **NOT** archived and remain in the root directory for production use:
- `index.html` - Main application
- `script.js` - Active JavaScript code
- `styles.css` - All styling
- `recipes.json` - Streamlined recipe data (3,203 lines)
- `README.md` - Main project documentation
- `assets/` - Family photos for carousel
- `MeeMaw Recipe Book Image File/` - Original cookbook page images
- `scripts_archive/` - Already organized development scripts

## Notes

These archived files are preserved for:
1. Historical reference
2. Ability to recreate conversion processes if needed
3. Rollback capability if issues arise
4. Understanding the evolution of the project

Most files in this archive are no longer needed for day-to-day operation but may be useful for future modifications or troubleshooting.
