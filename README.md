# MeeMaw Carle's Recipe Book

A digital cookbook preserving the treasured recipes of Alma Finken Carle, with recipes passed down from Clara Keller Finken, Caroline Nagel, Lizzie Walters, and family friends.

## About This Collection

These recipes were compiled by Wilfred Carle on April 8, 1995. Many have been modified to use:
- Egg substitutes instead of eggs
- Vegetable oils instead of lard, shortening, or animal fats
- Adjusted for taste preferences

## How to Use

1. Open `index.html` in any modern web browser
2. Use the search box to find recipes by name or ingredient
3. Filter by category using the dropdown menu
4. Click on any recipe card to view full ingredients and instructions
5. Press ESC or click the X to close the recipe detail view
